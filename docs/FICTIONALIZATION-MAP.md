# Fictionalization Map (D2 legal packet — ROADMAP P4.8)

The commercial roster and script catalog use fictional, era-authentic personas and titles. Real people and events remain only as public-record references in historical event text (news of the era), never as hireable characters or player-ownable productions.

**Scope applied:** all 173 talent-roster entries renamed (stats, availability windows, and archetypes preserved); ~170 real-surname references scrubbed from roster biographies; 91 verbatim real film titles renamed to pastiche; the four rival studios renamed to pastiche (Meridian Grand / Whitmore Brothers / Regency Radio / Pinnacle) with internal ids unchanged; rival talent pool and the production-system placeholder cast fictionalized.

**For counsel review:** (1) roster biographies still describe career shapes inspired by real figures — archetypes, not biographies, but flag any entry reading as identifiable; (2) historical event text intentionally names real public figures in a news-reporting frame; (3) the Academy Awards HISTORICAL_WINNERS table displays real historical facts as almanac content.
