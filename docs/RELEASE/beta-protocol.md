# Beta protocol (ROADMAP P7.3) — the outstanding human gate

**Recruit** 10–20 players (tycoon-genre familiarity mix). Fresh browser
profile; no coaching beyond the page copy.

**Session**: play ≥60 minutes, then complete the form. Ask them to
narrate stuck moments aloud (or note timestamps).

**Form (mirrors the audit scorecard, 1–5 each + free text):**
1. First hour: did you understand the loop? First informed decision at
   what minute? One win + one setback felt?
2. Economy: did money feel scarce/fair/exploitable? Did you find a
   strategy that trivialized it? Describe it.
3. History: name two events that changed your plans (not just appeared).
4. UI: anything unreadable/unreachable? Where did you scroll-hunt?
5. Audio: music noticed? Era changes noticed? Annoyances?
6. Bugs: paste any COPY BUG REPORT output (error screen button).
7. Would you pay $19.99? What's missing at that price?
8. Want to keep playing? Why/why not?

**Exit gate (BUILD-PLAN)**: ≥10 completed forms; blind first-hour pass
4/5; every S0/S1-class report fixed or waived with rationale; one
sim-gated tuning cycle applied on the data.
