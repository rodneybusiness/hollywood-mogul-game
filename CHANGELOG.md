# Changelog

## v3.0.0 — "The Golden Age" (2026-07)

The commercial release build. Highlights of the 2.x → 3.0 program
(full engineering record: docs/AUDIT-REPORT.md → docs/ROADMAP.md):

- **Scope**: the campaign is now the deep 1933–1949 arc, ending in a
  scored, epilogue-driven finale at January 1950.
- **The game is winnable, dangerous, and tuned** — a permanent economy
  simulation gates every balance change (passivity dies, skilled play
  compounds, no dominant strategy, no early money-cap).
- **History has teeth**: 151 historical events all carry mechanical
  consequences (genre currents, cost climates, the Paramount Decree,
  a real pre-Code window, HUAC choices with lasting fallout).
- **All studio personnel and film titles are fictional** era-authentic
  creations; the history is real (docs/FICTIONALIZATION-MAP.md).
- **Sound**: a generative period score with era transitions, plus full
  action SFX; volume/mute honored.
- **Replayability**: per-run script markets, shifting talent debuts,
  7 scenarios, conduct-aware endings, obtainable achievements.
- **Presentation**: title screen, studio naming, pinned action bar,
  animated money HUD, fully offline (self-hosted fonts).
- **Stability**: unified simulation pipeline, save format v3 with
  migration from v2, receivership instead of instant bankruptcy,
  global error boundary with copyable bug reports.
