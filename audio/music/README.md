# Background Music Directory

Place all background music files here. The audio system expects the following files:

## Era-Based Music (4 files)

1. `1930s-jazz-theme.mp3` - Pre-Code Era (1933-1934)
2. `1930s-orchestral.mp3` - Production Code Era (1934-1941)
3. `1940s-patriotic.mp3` - War Era (1941-1945)
4. `1940s-optimistic.mp3` - Post-War Era (1945-1949)

## Contextual Music (4 files)

5. `tension-low-finances.mp3` - Plays when cash is low
6. `success-fanfare.mp3` - Plays for major achievements
7. `crisis-dramatic.mp3` - Plays during major decisions
8. `menu-theme.mp3` - Plays on loading/menu screens

## Royalty-Free Music Sources

- **FreePD.com** - Public domain music
- **Incompetech.com** - Kevin MacLeod's royalty-free library
- **Free Music Archive** - Various Creative Commons tracks
- **YouTube Audio Library** - Free background music

## Format Requirements

- **Format**: MP3 or OGG
- **Bitrate**: 128-192 kbps
- **Sample Rate**: 44.1 kHz
- **Channels**: Stereo
- **Duration**: 2-4 minutes (looping tracks)

See `/audio/AUDIO_GUIDE.md` for detailed implementation guide.
